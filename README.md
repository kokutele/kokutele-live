# kokutele-live

## development work-flow

### pre

```
$ npm install
```

### developemt

```
$ npm start
```

### build

```
$ npm build
```

### test

```
$ npm test
```

## Environments

* REACT_APP_SERVER
  - server name of signaling server
* REACT_APP_PORT
  - server port of signaling server
* REACT_APP_USE_TLS
  - `true` when using TLS on signaling server ( https and wss )
